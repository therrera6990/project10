## Lab Questions

### What is difference between edge and level sensitive circuits?


Edge level circuits only trigger on the signal transitions such as rising or failing edges, but level sensative circuits trigger when the input remains high or low which is more prone to glitches

### Why is it important to declare initial state?

Because if we do not declare an initial state the flip flop will be undefined or random.


### What do edge sensitive circuits let us build?

It allows us to build registers, counters, basic memory systems. It helps us create reliable, synchronous digiital circuits overall.
